ZETASIM Web API: HTTP requests

Returning all the categories with the businesses belonging to each category

o /api/businesscategories

Returning a single Category

o /api/businesscategories/id

Returning List of businesses that belongs to a category

o /api/businesscategories/id/businesses

Returning a business that belong to a category

o /api/businesscategories/id/businesses/id

Returning all busineses without accesss categories: Retuns same as in (3)

o /api/businesses

Returing single business: Returns same as in (4)

o /api/businesses/id

Allowing inclusion of associated resources

o /api/businesscategories/1?fileds=id,description

Pagination

o /api/addresses?page=2&pagSize=3

Inclusion of associated resources

Support for Field Selection:
o /api/businesscategories/1/businesses?fields=id,description
Association and Data shapping
o http://localhost:60068/api/businesscategories?fields=businesses o http://localhost:60068/api/businesscategories?fields=id,description,businesses
Association using fields on the query string on businesses for addresses
o http://localhost:60068/api/businesses?fields=addresses o /api/businesses?fields=businessName,openingHour,closingHour,addresses (with More Fields). • http://localhost:60068/api/businesses?fields=id,dateCreated,businessName,businessType,description,addresses http://localhost:60068/api/businesscategories?fields=id,categoryName,description,businesses

Create New Category:

POST : http://localhost:60068/api/businesscategories {"categoryName": "SPAs", "description": "quam vel sapien imperdiet ornare."}

Update a Category:

http://localhost:60068/api/businesscategories/102 { "id": 102, "categoryName": "Accuressure updated", "description": "quam vel sapien imperdiet ornare." }// include id in the update body
